# 📅 Система скачивания расписания МАИ

Полноценная система для получения и удобного скачивания расписания занятий МАИ в формате .ics для импорта в календарь.

## 🏗 Архитектура проекта

```
rasp/
├── 📁 Python Backend
│   ├── group_parser.py      # Парсер групп из сайта МАИ
│   ├── get_rasp.py         # Получение расписания для группы
│   ├── main.py             # Основной скрипт для обновления данных
│   └── groups/
│       └── groups.json     # Данные о всех группах
├── 📁 React Frontend
│   ├── frontend/           # React + TypeScript приложение
│   │   ├── src/           # Исходный код
│   │   ├── public/        # Статические файлы
│   │   │   ├── groups.json
│   │   │   └── rasp/      # Файлы расписания (.ics)
│   │   └── package.json
│   └── README.md
├── 📁 Данные
│   └── rasp/              # 1188+ файлов расписания (.ics)
└── 📄 Документация
    ├── README.md
    └── update-frontend.sh # Скрипт обновления данных
```

## 🚀 Быстрый старт

### 1. Обновление данных расписания

```bash
# Обновить все данные (группы + расписания)
python main.py

# Или использовать удобный скрипт
./update-frontend.sh
```

### 2. Запуск React приложения

```bash
cd frontend
npm install
npm run dev
```

Откройте http://localhost:5173 в браузере.

### 3. Деплой на GitHub Pages

Следуйте инструкциям в [frontend/DEPLOYMENT.md](frontend/DEPLOYMENT.md)

## 🔧 Компоненты системы

### Python Backend

- **group_parser.py**: Парсит список всех групп с сайта МАИ
- **get_rasp.py**: Получает расписание конкретной группы и конвертирует в .ics
- **main.py**: Координирует процесс обновления всех данных

### React Frontend

- **Современный UI**: Красивый интерфейс с градиентами и анимациями
- **TypeScript**: Полная типизация для надежности
- **Адаптивный дизайн**: Работает на всех устройствах
- **Статические файлы**: Не требует бэкенда

## 📊 Данные

## 🔄 Обновление данных

Данные расписания обновляются **автоматически** через GitHub Actions:

### 🤖 Автоматическое обновление:
- **Ежедневно в 6:00 UTC** (9:00 МСК)
- **Полностью автоматический процесс**
- **Коммитит только измененные файлы**
- **Автоматический деплой на GitHub Pages**

### 🎯 Ручной запуск:
```bash
# Через GitHub Actions (рекомендуется)
gh workflow run "Update Schedule Data"

# Локальное обновление
python main.py
./update-schedule.sh
```

### 📊 Что обновляется:
1. **Парсинг групп**: Извлекает актуальный список групп с сайта МАИ
2. **Получение расписаний**: Скачивает расписание для каждой группы
3. **Конвертация**: Преобразует в формат .ics
4. **Синхронизация**: Обновляет файлы в React приложении
5. **Деплой**: Автоматически публикует обновления

### 📈 Мониторинг:
- Статус обновлений в разделе **Actions**
- Информация о последнем обновлении на главной странице
- Подробные логи выполнения
