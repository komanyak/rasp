# 📅 Скачивание расписания МАИ

React приложение для удобного скачивания расписания занятий МАИ в формате .ics для импорта в календарь.

## 🚀 Возможности

- Выбор института из списка всех институтов МАИ
- Выбор курса и уровня образования
- Выбор конкретной группы
- Скачивание расписания в формате .ics
- Красивый и адаптивный интерфейс
- Работа с статическими файлами (без бэкенда)

## 🛠 Технологии

- React 19
- TypeScript
- Vite
- CSS Modules
- Адаптивный дизайн с множественными брейкпоинтами
- CSS3 с градиентами и анимациями
- Кастомные React хуки
- Модульная архитектура компонентов
- GitHub Pages для деплоя

## 🚀 Деплой на GitHub Pages

1. **Обновите homepage в package.json:**
   ```json
   "homepage": "https://yourusername.github.io/rasp"
   ```

2. **Инициализируйте git репозиторий:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```

3. **Добавьте удаленный репозиторий:**
   ```bash
   git remote add origin https://github.com/yourusername/rasp.git
   git push -u origin main
   ```

4. **Деплой на GitHub Pages:**
   ```bash
   npm run deploy
   ```

5. **Настройте GitHub Pages в настройках репозитория:**
   - Перейдите в Settings → Pages
   - Выберите Source: "Deploy from a branch"
   - Выберите Branch: "gh-pages"
   - Нажмите Save

## 🔄 Обновление данных

Для обновления списка групп и расписания:

1. Запустите Python скрипты в корневой папке:
   ```bash
   python main.py
   ```

2. Скопируйте обновленные файлы:
   ```bash
   cp groups/groups.json frontend/public/
   cp rasp/*.ics frontend/public/rasp/
   ```

3. Пересоберите и задеплойте:
   ```bash
   npm run build
   npm run deploy
   ```

## 📱 Адаптивные брейкпоинты

Приложение оптимизировано для всех размеров экранов:

### Мобильные устройства
- **320px+** (XS): Очень маленькие экраны
- **480px+** (SM): Маленькие мобильные
- **576px+** (MD): Средние мобильные

### Планшеты
- **768px+** (LG): Маленькие планшеты
- **992px+** (XL): Большие планшеты

### Десктоп
- **1200px+** (XXL): Маленькие десктопы
- **1400px+** (XXXL): Большие десктопы
- **1600px+** (XXXXL): Очень большие экраны

### Особенности адаптивности:
- Динамическое изменение размеров шрифтов
- Адаптивные отступы и размеры элементов
- Оптимизация для сенсорных экранов
- Поддержка темной темы (prefers-color-scheme)
- Grid-раскладка для больших экранов
